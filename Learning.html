Java Basics - Learn with Diagrams

Java is a popular object-oriented programming language. Let's start with the basics.

1. Java Program Structure

A basic Java program consists of a class, a main method, and statements inside the method.

            
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

2. Variables and Data Types

Java has different data types for storing values:

  • int - Integer numbers
  • double - Decimal numbers
  • char - Single character
  • boolean - True or False

3. Conditional Statements

Conditional statements allow decision-making in code:

            
if (x > 0) {
    System.out.println("Positive Number");
} else {
    System.out.println("Negative Number");
}

4. Loops in Java

Loops are used to repeat a block of code multiple times.

            
for (int i = 0; i < 5; i++) {
    System.out.println("Iteration: " + i);
}

5. Functions in Java

Functions (methods) allow code reusability.

            
public static int add(int a, int b) {
    return a + b;
}
